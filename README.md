   - `[YOUR FIELD]`
   - `[YOUR MAIN RESEARCH AREA]`

   - `[TOPIC TWO]`

   - `your-email@example.com`

2. Add your CV to `assets` and call it `resume.pdf`.

3. Replace every `href="#"` with a real web link or a PDF path.
   Example:
   `href="assets/my-thesis.pdf"`

4. Open `index.html` in your browser to test the website.
